📱 인스타그램 홈페이지 분석

🎨 주요 레이아웃 구조

  • 헤더: 로고, 검색창, 알림/메시지/북마크 아이콘
    <header><logo/><input type="search"/><icons/></header>
  • 사이드바 (왼쪽): 네비게이션 메뉴 (홈, 동영상, 메시지, 알림, 만들기, 프로필)
    <nav class="sidebar" style="flex: 2;"><a href="#">홈</a>...</nav>
  • 메인 콘텐츠: 중앙 피드 (포스트 단일 열 배치)
    <main class="feed" style="flex: 6;"><article>...</article></main>
  • 오른쪽 사이드바: 온라인 친구, 추천 프로필
    <aside class="sidebar-right" style="flex: 2;"><user/>...</aside>

🛠️ 주요 HTML 요소

  • <header> - 상단 헤더 영역
    <header class="instagram-header"> ... </header>
  • <nav> - 사이드 네비게이션
    <nav class="sidebar"> <a class="sidebar-item">...</a> </nav>
  • <main> - 메인 피드 영역
    <main class="main-feed"> ... </main>
  • <article> - 각 포스트 컨테이너
    <article class="post"> ... </article>
  • <img> - 포스트 이미지
    <img src="post.jpg" alt="포스트 이미지">
  • <button> - 좋아요, 댓글, 공유 버튼
    <button class="post-icon-btn"> <svg>...</svg> </button>

🎯 CSS 주요 기법

  • Flexbox: 헤더, 네비게이션 정렬
    .sidebar { flex: 2; display: flex; flex-direction: column; }
    .main-feed { flex: 8; }
  • Grid: 포스트 피드 레이아웃
    .feed { display: grid; grid-template-columns: 1fr; gap: 16px; }
  • Position: 절대 위치 요소 배치
    .post-options { position: absolute; top: 10px; right: 10px; }
  • Hover 효과: 버튼 인터랙션
    .post-icon-btn:hover { opacity: 0.7; transform: scale(1.1); }
  • Transition: 부드러운 애니메이션
    .sidebar-item { transition: all 0.2s ease; }

🎬 릴스/긴영상 구성 요소

  • 비디오 플레이어: <video> 태그, 자동 재생, 음소거 상태
    <video autoplay muted loop width="100%"><source src="video.mp4"></video>
  • 사용자 정보: 프로필 사진, 이름, 팔로우 버튼 (상단 좌측)
    <div class="user-header"><img class="profile"><span>username</span></div>
  • 설명 텍스트: 캡션, 해시태그, 멘션 (좌측 하단)
    <p class="caption">설명 텍스트 #해시태그 @멘션</p>
  • 음악/사운드: 뮤직 노트 아이콘, 음악 정보 표시
    <span class="music-info"> ♪ 음악이름 - 아티스트 </span>
  • 상호작용 버튼: 우측 세로 배치 (좋아요, 댓글, 공유, 북마크, 옵션)
    <div style="flex-direction: column;"><button>♡</button><button>💬</button></div>
  • 조회수/통계: 좋아요 수, 댓글 수 표시
    <div class="stats">좋아요 <strong>1,234</strong> 댓글 <strong>56</strong></div>

🎯 아이콘 상세 분석

좋아요 기능: 클릭 시 상태 토글 (좋아요/취소)
CSS: hover 시 scale(1.1), 클릭 후 색상 변화 (#E1306C)
적용: 개수 증감, DB 저장, 알림 전송
댓글 기능: 클릭 시 댓글 창 열기 (모달)
CSS: hover 시 opacity 0.7
적용: 새 페이지 또는 모달에서 댓글 입력 폼 표시
공유 기능: 클릭 시 공유 메뉴 표시
CSS: hover 시 배경색 변화
적용: 메시지 전송, SNS 공유, 링크 복사

💡 아이콘 호버 효과 팁

  • 호버 시 시각적 피드백: 사용자가 아이콘을 클릭할 수 있다는 것을 시각적으로 알려줌
    .icon:hover { opacity: 0.7; transform: scale(1.1); cursor: pointer; }
  • 트랜지션 효과: 부드러운 변화로 사용자 경험 향상
    .icon { transition: all 0.2s ease; }
  • 라벨 표시: 호버 시 텍스트 라벨을 보여줘서 명확한 기능 설명
    .icon-label { display: none; } .icon:hover .icon-label { display: block; }
  • 색상 변화: 특정 색상(예: #E1306C)으로 강조해서 상호작용 표시
    .icon:hover { fill: #E1306C; stroke: #E1306C; }

🔄 아이콘 클릭 시 동작 흐름

  • 좋아요 버튼:
    사용자 클릭 → 상태 변경 (빈 하트 ♡ ↔ 채운 하트 ♥) → 색상 변화 (#E1306C) → 개수 증가 → DB 저장 → 알림 전송
    클릭 → 토글(toggle) → 색상 변경 → 카운트 업데이트 → 서버 요청
  • 댓글 버튼:
    사용자 클릭 → 댓글 모달/페이지 열기 → 텍스트 입력 → 제출 → 댓글 추가 → DB 저장
    클릭 → 모달 표시 → 사용자 입력 → 검증 → 서버 저장 → 새로고침
  • 공유 버튼:
    사용자 클릭 → 공유 옵션 메뉴 표시 (메시지, 스토리, 복사) → 선택 → 해당 플랫폼으로 이동/실행
    클릭 → 드롭다운 메뉴 표시 → 옵션 선택 → 링크 생성 → 공유 실행
  • 저장 버튼:
    사용자 클릭 → 상태 변경 (미저장 ☐ ↔ 저장됨 ☑) → 개인 컬렉션에 추가 → DB 저장
    클릭 → 아이콘 변경 → 컬렉션 추가 → 서버 저장

📚 HTML & CSS 학습 정리

🌐 HTML 학습 내용 (01 ~ 19)

  • 01번: HTML 소개 - 웹 페이지의 구조를 담당하는 마크업 언어
  • 02번: HTML 기본 구조 - DOCTYPE, html, head, body 태그의 역할
  • 03번: 제목 태그 - h1~h6로 텍스트의 계층 구조 표현
  • 04번: 문단 태그 - p, br, hr로 텍스트 블록 생성
  • 05번: 텍스트 서식 - strong, em, mark로 강조 표현
  • 06번: 링크 태그 - a href로 다른 페이지 연결
  • 07번: 이미지 태그 - img src로 이미지 삽입
  • 08번: 멀티미디어 - audio, video 태그로 미디어 재생
  • 09번: 순서 없는 리스트 - ul, li로 항목 나열
  • 10번: 순서 있는 리스트 - ol, li로 순번 표시
  • 11번: 리스트 활용 - 자기소개 웹사이트 구축 프로젝트
  • 12번: 테이블 기본 - table, tr, td로 표 생성
  • 13번: 테이블 고급 - thead, tbody, colspan, rowspan
  • 14번: 테이블 실습 - 시간표, 가격표 등 실제 표 제작
  • 15번: 폼 기본 - form, input, label로 입력 양식 생성
  • 16번: 입력 요소 - text, password, email, checkbox, radio 등
  • 17번: 폼 고급 - select, textarea, button으로 다양한 입력
  • 18번: 폼 실습 - 로그인, 회원가입 페이지 제작
  • 19번: 종합 프로젝트 - 자기소개 및 포트폴리오 웹사이트
